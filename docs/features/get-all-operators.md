# Get All Operators

## Description

___
This feature returns all the operators and their information.

* URL: `/Operator`
* Method: `GET`

___

* Example Response:

    ```json
      [
        {
            "id": 1,
            "name": "Eli Gandin"
        },
        {
            "id": 2,
            "name": "Fiona Green"
        },
        {
            "id": 3,
            "name": "George Harris"
        }
      ]
    ```

* Response Codes:
    * `200` - Success
    * `500` - Internal server error
